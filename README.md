# C++ Boilerplate

## Installation

Generate build files with:

```
$ git clone git@github.com:vaalentin/cpp-boilerplate.git
$ cd cpp-boilerplate
$ mkdir build
$ cd build
$ cmake ..
```

## Usage

From the build folder:

```
- make # build
- ./main # execute project
```
